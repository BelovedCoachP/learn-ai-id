Setup (One-Time)

Install the Stack

Everything runs locally through Claude Code. No monthly video software subscriptions.
  1. Install Remotion via Claude Code terminal (JavaScript-based video rendering library)
  2. Install the Claude Chrome Extension (needed for pulling website assets for product launch videos)
  3. Install Blotato MCP connector for auto-publishing to social platforms after render
  4. Optionally install the Python tool in Cowork for script-based animated explainers
Video Types You Can Generate

Type 1: Education Explainer

Animated motion graphics with text overlays. Perfect for course content.
  1. Give Claude a topic or script prompt
  2. Claude generates a Remotion component with motion graphics and animated text
  3. Review the first render. Iterate via storyboards if needed
  4. Export final video file locally
Your angle: WCAG explainer videos for LearnAIID Academy on Payhip. "What is alt text and why it matters" as an animated 60-second clip. Sell the course, give the explainer away as marketing.

Type 2: Product Launch / Demo

Claude pulls assets from your website and builds a simulated product walkthrough.
  1. Give Claude your product URL (e.g., learnaiid.com or a Payhip listing)
  2. Chrome Extension pulls screenshots, logos, and copy directly from the site
  3. Claude generates a simulated interactive demo video via Remotion
  4. Render and export locally
Your angle: Demo videos for each Payhip micro-course listing. "Here's what you get inside the Document Accessibility course" as a 30-second product walkthrough. Also works for pitching Classfindr to new institutions.

Type 3: Testimonial / Social Proof

Pulls Google reviews and animates them as social proof clips.
  1. Point Claude at your Google Business listing (via Chrome Extension or local export)
  2. Claude pulls review text automatically
  3. Remotion renders them as animated social proof video clips
  4. Export locally, sized for Instagram Reels or TikTok
Your angle: When you start collecting LearnAIID or QM workshop feedback, turn the best quotes into animated testimonial reels. Instant credibility content.

Type 4: Edit Existing Video

Add captions, titles, and web screenshots to existing footage. No manual editing.
  1. Provide Claude with an existing clip (talking-head, AI avatar, screen recording)
  2. Claude auto-adds dynamic captions, titles, and web screenshot overlays via Remotion
  3. Outputs a fully edited video file
Your angle: Your VCCS road trip raw footage. Your QM workshop recordings. Add branded captions and LearnAIID lower thirds without opening Filmora.

Type 5: Data Visualization

Animated charts and infographics from any dataset or study.
  1. Give Claude a link to a study, dataset, or spreadsheet
  2. Claude reads the data and generates animated charts and infographics via Remotion
  3. Export as video or GIF
Your angle: VCCS compliance progress across 23 colleges as an animated bar chart. DOJ Title II deadline countdown with stats. Killer for LinkedIn and conference slides.
